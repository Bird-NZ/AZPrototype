FAQ

Frequent questions about operating and explaining AZ Prototype.

Is this a site generator project?

No. This scaffold is intentionally plain static HTML/CSS/JS so it can deploy cleanly to GitHub Project Pages with minimal moving parts.

Can I use this with GitHub Pages under a project subpath?

Yes. The site uses relative asset and page links, so it works at https://<user>.github.io/AZPrototype/.

Do I need GitHub auth?

Only if you are using the copilot or github-models provider path. azure-openai skips the GitHub auth flow.

When should I use templates?

Use them to accelerate common workloads. Ignore them when your architecture is unusual enough that the template would just become cleanup debt.

What is the safest operator flow?

init → design → build → deploy --dry-run → inspect → deploy.

How do I preview locally?

python3 -m http.server 4173 -d az-prototype/site
