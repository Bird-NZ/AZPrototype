What AZ Prototype is

az prototype is an Azure CLI extension that turns requirements into an Azure prototype through a structured four-stage flow: init, design, build, and deploy.

  • Supports GitHub Copilot, GitHub Models, and Azure OpenAI
  • Uses a built-in team of specialized agents
  • Generates Terraform or Bicep plus application code and docs
  • Helps with architecture, governance, QA, monitoring, and cost analysis

Recommended learning path

  1. Read Getting started
  2. Run the Quickstart
  3. Follow the full Init → Design → Build → Deploy walkthrough
  4. Customize with templates and prototype.yaml
  5. Use deployment and dry-run guidance before real Azure rollout
