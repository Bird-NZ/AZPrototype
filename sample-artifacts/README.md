# Sample Artifacts for `az prototype design --artifacts`

This folder contains realistic sample inputs you can point AZ Prototype at during demos.

Included artifact types:
- markdown brief
- discovery notes text file
- slide-outline document
- CSV issue list
- JSON non-functional requirements
- sample PDF brief
- screenshot mock (SVG)

Example usage:

```bash
az prototype design --artifacts ./sample-artifacts --context "Use these materials to produce a pragmatic Azure prototype design."
```
